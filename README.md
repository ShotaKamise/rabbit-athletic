# ぴょんぴょんラビット 🐰

ジャンプと移動だけでゴールの旗を目指す、ガラケー風アクションゲーム（全10ステージ）。

## 遊び方
- **PC**: ←→ で移動、Z / スペースでジャンプ、R でやりなおし、M で BGM ON/OFF
- **スマホ**: 画面のボタンで操作

## 障害・ギミック
- 🔥 炎・🐦 トリ・🌊 水に触れるとミス
- 🟦 青ブロック: 乗ると一定時間で消える
- 🟥 赤スイッチ: 乗ると紫ブロックの実体が入れ替わる

## ローカルで遊ぶ
`index.html` をブラウザで開くだけで動きます。

## ホーム画面に追加（アプリとして使う）
GitHub Pages などの https 上で開くと PWA として動作し、
スマホの「ホーム画面に追加」でアプリのように全画面起動＆オフラインで遊べます。
